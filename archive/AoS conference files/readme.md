# AoS conference 2026

Files used in my presentation for the 2026 AoS conference

* [cha_probabilities.csv](cha_probabilities.csv): this is the dataset that I read into the package. It contains all settings for creating the dataset
* [mock_cha_data.csv](mock_cha_data.csv): this is the final dataset after: 
    * running functions in the package 
	* post-processing to add respondent ages and counties of residence
	* introducing deliberate errors
* [cha_codebook.xlsx](cha_codebook.xlsx): the codebook that describes the data
* [cha_survey.doc](cha_survey.doc): the survey instrument in Microsoft Word format
