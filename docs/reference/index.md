# Package index

## Multiple variable creators

Functions that calulate sets of variables

- [`select_categorical_set()`](https://ajstamm.github.io/toysurveydata/reference/select_categorical_set.md)
  : Create a set of single-selection categorical variables
- [`select_many_set()`](https://ajstamm.github.io/toysurveydata/reference/select_many_set.md)
  : Create a set of multiple-selection variables
- [`select_continuous_set()`](https://ajstamm.github.io/toysurveydata/reference/select_continuous_set.md)
  : Create a set of single-selection continuous variables
- [`select_dates_set()`](https://ajstamm.github.io/toysurveydata/reference/select_dates_set.md)
  : Create a set of single-selection date variables

## Single variable creators

Functions that calulate single variables

- [`sample_one()`](https://ajstamm.github.io/toysurveydata/reference/sample_one.md)
  : Create a random vector of responses
- [`sample_many()`](https://ajstamm.github.io/toysurveydata/reference/sample_many.md)
  : Calculate multiple booleans
- [`sample_ip()`](https://ajstamm.github.io/toysurveydata/reference/sample_ip.md)
  : Create fictional IP addresses

## Data manipulators

Functions that derive variables or add error to variables

- [`sample_age()`](https://ajstamm.github.io/toysurveydata/reference/sample_age.md)
  : Create fictional ages based on birth date and survey date
- [`sampling_error()`](https://ajstamm.github.io/toysurveydata/reference/sampling_error.md)
  : Add sampling data errors
