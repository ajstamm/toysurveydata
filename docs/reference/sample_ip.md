# Create fictional IP addresses

Create fictional IP addresses

## Usage

``` r
sample_ip(count)
```

## Arguments

- count:

  integer; number of responses

## Examples

``` r
sample_ip(count = 5)
#>   id      ip_address
#> 1  1 229.195.254.189
#> 2  2   102.229.23.54
#> 3  3  111.200.78.170
#> 4  4    47.196.76.61
#> 5  5    159.4.129.42
```
