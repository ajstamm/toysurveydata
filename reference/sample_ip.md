# Create fictional IP addresses

Create fictional IP addresses

## Usage

``` r
sample_ip(count)
```

## Arguments

- count:

  integer; number of responses

## Examples

``` r
sample_ip(count = 5)
#>   id      ip_address
#> 1  1  173.101.229.23
#> 2  2  151.110.200.78
#> 3  3   204.46.196.76
#> 4  4    63.158.4.129
#> 5  5 229.195.254.189
```
